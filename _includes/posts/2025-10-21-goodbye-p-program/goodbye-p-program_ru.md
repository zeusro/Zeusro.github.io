Например, настройка этого GitHub Action заняла бы у меня 2-3 дня раньше. На этот раз, с помощью OpenAI, это было завершено в течение 3 часов.

## Hello, GitHub Action

Процесс настройки очень прост:
1. Измените настройки pages-GitHub Pages для [p-program.github.io](https://github.com/p-program/p-program.github.io) на значения по умолчанию, оставьте Custom domain пустым
1. Решите **Dependabot alerts** в проекте исходного кода
1. Добавьте `.github/workflows/deploy.yml` в проект исходного кода согласно подсказкам

После настройки вам больше не нужно экспортировать статические HTML-артефакты, DNS не нужно изменять (предотвращение бесплатного использования), и он автоматически обновляется при каждом событии push — просто идеально, стоит попробовать.

```yml
on:
  push:
    branches:
      - mster      # Если ваша основная ветка — master / main / mster, настройте на правильное имя ветки
  # Сохранить ручной триггер
  workflow_dispatch:
```

## Решение некоторых исторических проблем

```

Error: The updater encountered one or more errors.

For more information see: https://github.com/p-program/readme/network/updates/1130827200 (write access to the repository is required to view the log)
```

Проверки безопасности Dependabot в основном нацелены на систему цепочки поставок проекта — некоторые зависимости не были обновлены до безопасных версий. Исправьте согласно подсказкам или обновите вручную.

Проработав в информационных технологиях много лет, я всегда имел противоречивые мнения об этой отрасли. Но в целом я в основном против.
Технологическое развитие не освободило производительность по-настоящему; вместо этого оно сделало людей инструментами для количественных расчётов.

Так же, как в 2020 году, когда я предложил [OOOS](https://www.bullshitprogram.com/one-open-operating-system/), используя [ИИ как интеллектуальный API-шлюз](https://www.bullshitprogram.com/the-seed-of-robot/), моё первоначальное намерение состояло в том, чтобы использовать ИИ для быстрой реализации человеческого творчества.

Но, как видно сегодня, большинство компаний используют его как инструмент увольнения, что действительно раздражает.
