## Парадигма базы данных — ядро ETCD

Лимонад бедняка — это окончательно доставленный продукт. Поток данных пользовательских запросов противоположен построению распределённой системы.

Для информационных систем база данных (ETCD) — это базовая инфраструктура. И парадигма базы данных — это ядро ETCD.

ETCD хранит этот чек структурированным способом (object).

Mixue Ice Cream & Tea (Kubernetes) поможет ей изготовить и доставить этот «лимонад бедняка» через следующие действия.

## Разрыв между реальностью и идеалом — цель доставки приложений

«Лимонад бедняка» в идеале мамы Хаохао (spec) и фактический (status) «лимонад бедняка» в руке — это два разных понятия. Коммерческое описание — «изображения только для справки».
Если такие платформы, как Meituan и Ele.me, считаются поставщиками облачных услуг (cloud provider), то штаб-квартира Mixue Ice Cream & Tea — это следующее звено во всём конвейере доставки (API server).

```draw
┌───────────────────────────────┐
│        Control Plane          │
│ ┌───────────────────────────┐ │
│ │ kube-apiserver            │ │
│ │ etcd (data store)         │ │
│ │ kube-scheduler            │ │
│ │ kube-controller-manager   │ │
│ │ cloud-controller-manager  │ │
│ └───────────────────────────┘ │
└───────────────────────────────┘
                 │
                 ▼
┌────────────────────────────────────────────┐
│               Worker Node                  │
│ ┌────────────┐  ┌────────────┐  ┌────────┐ │
│ │ kubelet    │  │ kube-proxy │  │ runtime│ │
│ └────────────┘  └────────────┘  └────────┘ │
│               └──► Pod(s) (containers)     │
└────────────────────────────────────────────┘
```

Штаб-квартира Mixue Ice Cream & Tea (API server/Control Plane Node) выдаёт задачи фактически выполняющим магазинам (Worker Node),
где менеджер магазина (kubelet) назначает конкретные задачи реализации, а конкретные сотрудники этих магазинов (Container Runtime) отвечают за окончательное производство продукта.

## Последняя миля — обязанность курьера

После изготовления продукта курьер должен выполнить окончательную доставку. Курьер на самом деле также является надзорным органом «лимонада бедняка» (бизнеса).

![img](/img/in-post/i-love-haohao/202312252211598563.jpg)

Во время транспортировки необходимо обеспечить целостность продукта (анализ наблюдаемости) и достичь согласованности между окончательной доставкой и требованиями клиента (ожиданиями).
